Справка

Добавление банков

Нажмите COIN вверху списка банков. Три способа:

  • Файл — выберите JSON файл COIN 0.9.0
  • Текст — вставьте JSON текст напрямую
  • URL — вставьте ссылку на COIN JSON файл

Импорт из скриншота (AI)

Если нет COIN файла — сделайте скриншот категорий в банковском приложении, загрузите в AI-чат (DeepSeek, ChatGPT, Gemini) с готовым промптом. AI вернёт JSON — вставьте через вкладку «Текст». Промпт и ссылки на AI-чаты — внизу модалки импорта.

Выбор категорий

Нажмите Выбрать на карточке банка. Отметьте нужные категории — выбор сохраняется на текущий месяц.

Лимит категорий

Некоторые банки ограничивают количество выбираемых категорий (например, 5 из 10). Лимит настраивается кнопками + − вверху списка. Строка прогресса показывает заполнение.

Категории «не в лимите»

Отметьте категорию как «не в лимите» (кнопка ∞ рядом) — она будет всегда выбрана и не учитывается в лимите. Используйте для кэшбэка «На все покупки» и подобных.

Блок категорий

Кнопка 🚫 рядом с категорией отмечает её как «не рекомендовать» — категория будет полупрозрачной и недоступной для выбора во всех банках. Алгоритм рекомендаций её проигнорирует. Нажмите повторно — разблокирует.

Доп. категории

Кнопка Доп. позволяет:

  • Новая — создать свою категорию с произвольным %
  • Доп. % — добавить бонусный % к существующей категории банка (показывается как «доп. +X%»)

Stackable cashback (доп.)

Если в банке есть две категории с одинаковым названием но разным % (например, «Транспорт» 12% и «Транспорт» 5%), они объединяются в одну строку. Кнопка +X% рядом позволяет вручную добавить бонусный % к основному.

  • Не активирован — показывается только основной (наивысший) %
  • Активирован — категория получает суммарный %, отмечается зелёной рамкой и тегом «доп. +X%»
  • Бонус активируется отдельно от выбора категории и не учитывается в лимите

Рамки карточек банков

  • Жёлто-золотая рамка — категории не выбраны. Нажмите «Выбрать» и отметьте нужные.
  • Цветная рамка — есть выбранные категории.
  • Яркая рамка — список категорий раскрыт.

Кнопки в карточке банка

  • Рекомендации — алгоритм автоматически выбирает категории с максимальным %, учитывая все ваши банки. Рекомендуемые категории выделены жёлтой полоской слева и жёлтой молнией.
  • Прошлый — скопировать выбор с прошлого месяца
  • Месяц — перейти к следующему месяцу (очищает выбор)
  • ↑ ↓ — переместить банк выше/ниже в списке

Обновление банка

Если импортировать COIN файл с названием уже существующего банка — предложит «Обновить категории» (сбросить выбор) или «Обновить и сохранить выбор» (сохранить совпадающие).

Что такое MCC?

MCC (Merchant Category Code) — 4-значный код, который банк присваивает каждому платежу по типу магазина. По MCC банк определяет, к какой категории относится покупка.

Зачем нужен справочник?

Названия категорий у банков разные. Справочник помогает выяснить, какой MCC у конкретного магазина и попадает ли он под кэшбэк.

Поиск

Введите код (например, 5812) или часть названия (например, ресторан) — все группы с совпадениями раскроются автоматически, совпадения подсветятся.

Связь с кэшбэком

Когда банк пишет «Рестораны 5%», он имеет в виду определённые MCC (обычно 5811, 5812 и др.). Если магазин зарегистрирован с другим MCC — кэшбэк может не начисляться. Это частая причина «не прошёл кэшбэк».

💡 Совет: перед крупной покупкой проверьте MCC магазина в чеке или выписке и сверьте с категориями кэшбэка вашего банка.

Смена темы

Кнопка 🌙 / ☀️ переключает между тёмной и светлой темой. Выбор сохраняется между запусками.

Экспорт / Восстановление

Все данные хранятся локально на устройстве. Для переноса на другое:

  • Экспорт — скачивает JSON файл со всеми банками, выборами и настройками
  • Восстановление — загружает ранее сохранённый файл

Очистка данных

Кнопка 🗑️ удаляет все банки, выборы и настройки. Действие необратимо (если нет экспорта).

Установка на телефон

Кнопка ⬇️ в шапке (когда доступна) добавляет приложение на главный экран как PWA. Работает оффлайн после первой загрузки.
